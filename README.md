Installation

cd laravel

composer install

создать базу данных

создать файл .env, в нём подкючиться к базе данных

php artisan migrate

Раскоментировать нужные сиды и выполнить. Папка database/seeders/DatabaseSeeder.php и запустить сиды на выполнение php artisan db:seed

php artisan serve
